# Bug Bounty Evidence Folder

This directory contains all the evidence and documentation for the security vulnerabilities discovered in `facebookincubator/isometric_pattern_matcher`.

## Contents

- **`Meta_Submission_Report.md`**: The bug report drafted specifically for submission to the Meta Bug Bounty program. It includes the Title, Description, Impact, and Reproduction Steps in their requested format.
- **`Technical_Bug_Report.md`**: The original, more detailed technical breakdown of the 3 vulnerabilities with deep code analysis and fix recommendations.
- **`poc_trigger_crash.cpp`**: A clean, standalone compilation C++ Proof of Concept script that instantiates the classes and artificially triggers the Out-Of-Bounds (OOB) `SIGABRT` vulnerability.
